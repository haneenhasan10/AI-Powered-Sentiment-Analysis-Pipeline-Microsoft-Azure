**Project Overview**

This project demonstrates an end-to-end AI solution for analyzing customer feedback. By leveraging Microsoft Azure AI Language Services, I processed dataset from Kaggle. The goal is to transform unstructured text into actionable business intelligence using a modern cloud-native approach.

**Tech Stack**

Cloud Platform: Microsoft Azure (Language Service)

Data Source: Kaggle (Amazon)

Data Format: CSV

Visualization: Power BI

Data Handling: Excel/JSON Transformation

**Workflow & Strategy**

Data Ingestion: Sourced a real-world dataset from Kaggle

AI Processing: Utilized Azure Language Studio (Pre-built Sentiment Analysis models) to process the text.

Data Extraction: Exported the AI results in JSON format.

Transformation: Structured the JSON output into a tabular format for analytical readiness.

Visualization: Designed a Power BI Dashboard to monitor the "Voice of the Customer".
